# amogus
Утилита для предзаписи на модеус

## Запуск
1. Клонируй репозиторий
`git clone https://github.com/Matvke/amogus.git`
2. Перейди в рабочую директорию
`cd amogus`
3. Создай виртуальное окружение
`python -m venv .venv`
4. Активируй виртуальное окружение
Windows
`.\.venv\Scripts\activate`
Linux/MacOS
`source .venv/bin/activate`
5. Установи проект
    1. Для разработки
        `pip install -e .`
    2. Для обычного использования
        `pip install .`
6. Получи код меню из URL меню выбора и впиши его в .env
`https://urfu.modeus.org/learning-path-selection/api/selection/menus/{menu_id}`
Нужно именно `menu_id`, а не вся ссылка.
> Пример .env фала можешь посмотреть в .env.example
7. Получи JWT токен и запиши его в .env
    1. Для этого перейди на страницу выбора
    2. Перейди в панель админа `f12`
    3. Вкладка Storage -> Session Storage
    4. Скопируй значение `id_token` в .env в поле token
8. Запусти TUI приложение
`amogus` 
> Можно явно указывать .env file `amogus --env-file ".env1"`   

## Структура проекта

```
.
├── amogus_tui_app/      # Основной пакет приложения
│   ├── api_methods.py   # Логика запросов к API
│   ├── disciplines.json # Данные о дисциплинах (локальные)
│   ├── entities.py      # Pydantic-модели и сущности
│   ├── settings.py      # Загрузка настроек и .env
│   └── tui_app.py       # Интерфейс на Textual
├── .env                 # Файл с секретами (создается вручную)
├── pyproject.toml       # Конфигурация сборки и зависимости
└── README.md
```

## Переменные .env
```
menu_id=<Твой UUID4>
token=<Твой JWT токен>
pick_time=<дата и время выбора предметов>
```
> Формат даты: %Y-%m-%d %H:%M например: "2026-03-31 16:33"

## Использование
> Символ `^` в интерфейсе приложения означает клавишу `Ctrl`.

Выбираешь команды сочетанием `ctrl+s`, сохраняешь их в файл (по желанию), запускаешь таймер для записи через `ctrl + v`

### Горячие клавиши

| Клавиша | Действие |
| :--- | :--- |
| `Arrows` (Стрелки) | Навигация по интерфейсу |
| `Tab` | Переключение со вкладок на рабочую область |
| `Ctrl + x` | Сохранить выбранные дисциплины в файл |
| `Ctrl + z` | Загрузить список команд из файла |
| `Ctrl + v` | Отправить запрос на запись в выбранные команды |
| `Ctrl + q` | Выйти из приложения |


### Работа с деревом (Вкладка Tree)

В этой вкладке вы управляете выбором конкретных команд:

*   **`Enter`** — Развернуть или свернуть ветку (ноду).
*   **`Ctrl + s`** — Выбрать текущую команду. 
    * *Выбранные команды автоматически отображаются во вкладке **Json** для предпросмотра.*
*   **`Ctrl + d`** — Удалить текущую команду.
